Leaflet Panel Layers

Leaflet Control Layers extended for group of layers and icons legend.
Other useful stuff for Web Mapping...

If any of these open source solutions help your work and saved you time consider sending a donation

Features

  • Grouping of layers
  • Custom layer item
  • Auto collapsible layers list
  • Infinite scrollable list of layers
  • Configurable panel from JSON
  • Grouping of multiple layers
  • Embeddable GeoJSON layers
  • Add layers dynamically

Examples

  • Icon layer
  • Collapsible panel
  • Group of layers
  • Group selector
  • Exlusive Group of layers
  • Dynamic layers list
  • Embedded JSON layers
  • Custom item panel
  • Layers thumbnails
  • Multiple panels
