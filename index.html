Leaflet Panel Layers

Leaflet Control Layers extended for group of layers and icons legend.
Other useful stuff for Web Mapping...

Features

  • Custom layer Icon
  • Support multiple Layer separator between base or over layers
  • Custom separator label
  • Resizable panel with the map

Examples

  • Points Of Interest
  • Groups of layers
